Descripción del proyecto
    Tragamonedas es un juego web sencillo creado con HTML, CSS y JavaScript. 
    Simula una máquina tragamonedas con símbolos de emoji y un sistema básico de crédito para jugar.

¿Qué hace?
    Muestra una interfaz de tragamonedas con tres casillas.
    Permite jugar pulsando el botón PLAY.
    Genera tres símbolos aleatorios en cada jugada.
    Si los tres símbolos coinciden, el jugador gana créditos.
    Si no coinciden, se descuenta un crédito.
    Muestra el saldo actual, las monedas como emojis y un mensaje de estado.

Características
    Diseño oscuro con estilo retro y emojis de frutas.
    Sistema de créditos inicial.
    Mensajes dinámicos: ¡Felicidades, has ganado! o ¡Sigue intentando!
    Prevención de juego sin créditos.

Archivos principales
    index.html — estructura de la página
    styles.css — estilos visuales
    maquina.js — lógica del juego

Uso
    Abrir index.html en un navegador.
    Presionar el botón PLAY.
    Observar los símbolos y el saldo.
    Repetir hasta ganar créditos o quedarse sin saldo.

Potenciales mejoras
    Añadir recarga de créditos.
    Incluir más símbolos y combinaciones de premio.
    Agregar animaciones al girar.
    Adaptar el diseño a móviles.
